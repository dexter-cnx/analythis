# CLI Reference

## analyze

```bash
analythis analyze <path-or-url> [--profile generic|web|backend|mobile|monorepo|library] [--focus auth,billing] [--output .analythis] [--format json|md|both] [--branch main] [--shallow]
```

## inspect

```bash
analythis inspect <path-or-url> [--output .analythis] [--branch main]
```

## export

```bash
analythis export <blueprint-json> [--to md|yaml] [--output .analythis-export]
```
