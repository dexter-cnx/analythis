# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-06

### Added
- Initial release
- Analyze local folders or remote Git repositories
- Generate `inventory.json` and `blueprint.json`
- Export blueprint as markdown or YAML
- Heuristic support for web, backend, mobile, library, and monorepo profiles
- Prompt-pack generation for refactoring, regeneration, core extraction, and onboarding
- CLI commands: `analyze`, `inspect`, `export`
